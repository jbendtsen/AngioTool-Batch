About

AngioTool was originally written by the Center for Cancer Research in 2011. It could only process one image at a time, which made it unsuitable for scientific research, despite the image processing features it possessed.

This version of AngioTool started by simply adding batch processing as a separate feature, where the user could select multiple folders instead of just one image at a time. However, after processing the 15th or so image, it would run out of memory and cease to function.

As such, the goal of the project was expanded, and gradually the entire program was rewritten piece by piece, including a new user interface, to allow for the batch processing feature to work as intended.

Features

Batch Processing

Select folders of images to process, the output image folder, and a spreadsheet to write to.

Live Preview

Analyze a single image. Supports panning and zooming, and changing the settings to update the preview in real time.

Settings

AngioTool 2.0 provides various customizable settings to tailor image analysis:

Lacunarity

Measure how much space is empty.

Thickness

Measure the average vessel thickness.

Fill Holes

Fill in gaps between vessels below a certain size.

Remove Particles

Remove particles below a certain size.

Max Hole Level

The brightest a hole can be before it shall be filled in.

Min Boxness

The lowest "boxness" value that a hole can have before it shall be filled in.

Boxness is defined as: min(holeWidth / holeHeight, holeHeight / holeWidth) * pixelCount / (holeWidth * holeHeight)

The more a hole resembles square, the closer the boxness value will be to 100% and the further it will be from 0%.

Min Length ÷ Area

The lowest area divided by length that a hole can have before it shall be filled in.

Skeletonizer

Finds the center line of each vessel.

Max Skeleton Steps

Cap the number of times an image can be iterated upon by the skeletonizer.

Each skeletonization iteration thins the edges of a shape by one or two pixels at a time.

Skeletonization is the stage in the analysis pipeline that takes the longest to complete, especially when there are large shapes that don't conform to either vessels, pathways or gaps between.

Resize Inputs

Scale up or down the image.

Measurement Scale

Scale up or down pixel measurements.

Vessel Diameters

Enter a list of target vessel thicknesses.

Vessel Intensity

Pixel brightness range.
